The plugin adds the Mapp Cloud Tag Integration code to your website and creates the data layer for your customer insights dashboards.

== Description ==

The Mapp Cloud plugin enables customers to quickly and easily set up the tracking environment needed to start analysing customer insights immediately. It creates the data layer used to feed the Mapp Intelligence customer insights and Mapp Acquire dashboards without requiring you to touch any code.

**Plugin Features**

* Adds the Mapp Cloud Tag Integration code to the footer of your website.
* Creates the standard data layer to start analyzing customer insights immediately.
* Enables customers to
 * add the information needed to initialize tracking.
 * exclude unwanted keys/information from the data layer.
 * exclude tracking of users logged in the WordPress admin panel.

**Mapp Cloud**

Mapp Cloud allows customers to gain in-depth customer insights and adjust marketing activities based on actual user behavior and data. == Installation ==

1. Add the plugin to your admin panel.
2. Add your Tag Integration ID /GTM ID. Leave the Tag Integration domain as is unless you host the tracking script on your own servers.
3. Optionally, exclude keys from the data layer.
4. Optionally, exclude users logged into the WordPress admin panel from being tracked via the pixel.
5. Optionally, add your Mapp Acquire script tag.
6. Save the plugin in WordPress.
7. Follow the instructions to set up tracking in Mapp Intelligence here: [WordPress Plugin](https://docs.mapp.com/v1/docs/en/wordpress-woocommerce).
